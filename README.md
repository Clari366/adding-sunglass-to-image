## 😎 Adding Sunglasses to Image Project

## ✨ Overview
This is a fun and creative image processing project that adds stylish sunglasses to photos. It transforms simple passport-size images into visually appealing and enhanced outputs using basic computer vision techniques.

## 🎯 Project Description
This project demonstrates how image processing can be used to detect a face and accurately overlay sunglasses. It is designed to be simple, interactive, and beginner-friendly, making it ideal for learning and experimentation.

## 🚀 Features
- Detects faces in an image  
- Automatically adds a sunglasses overlay  
- Works best with passport-size photos  
- Easy to customize with different sunglasses styles  
- Simple and beginner-friendly implementation  

## 🛠️ Technologies Used
- Python  
- OpenCV (for image processing)  
- NumPy (for array operations)  

## 📂 Project Structure
- main_sunglass.ipynb → Main implementation  
- sunglass.png → Sunglasses overlay image  
- abc.png → Sample input/output image  
- README.md → Project documentation  
- .gitignore → Ignored files  
- LICENSE → License information  

## ▶️ How to Use
1. Clone the repository  
2. Add your passport-size photo  
3. Run the notebook or script  
4. Get your photo with a cool sunglasses effect 😎  

## 🌟 Applications
- Learning basic image processing concepts  
- Practicing computer vision techniques  
- Creating fun photo filters
